Быстрый старт

1. Перейди в раздел Точки → нажми кнопку редактирования
2. Вставь Telegram Chat ID — туда будут приходить отчёты
3. Скопируй API ключ из карточки точки
4. На кассовом ПК запусти:
py -3.13 monitor.py --api-url https://твой-сайт.onrender.com --api-key <КЛЮЧ>
5. Говори что-нибудь — отчёт появится через ~15 секунд

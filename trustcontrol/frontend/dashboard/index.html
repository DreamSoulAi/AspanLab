О системе

TrustControl — это не инструмент слежки и не способ усложнить жизнь сотрудникам.

Когда бизнес растёт, владелец не может присутствовать везде. Клиенты уходят молча, не объясняя почему. Система помогает вернуть доверие и лояльность — замечая холодный тон, пропущенное приветствие, упущенную продажу. Хорошие сотрудники получают признание. Проблемы решаются до того, как клиент ушёл навсегда.

Быстрый старт

1. Раздел Точки → нажми ✏️ Изменить
2. Вставь Telegram Chat ID
3. Скопируй API ключ
4. На кассовом ПК запусти:
py -3.13 monitor.py --api-url https://сайт.onrender.com --api-key <КЛЮЧ>
5. Скажи что-нибудь — отчёт придёт через ~15 сек

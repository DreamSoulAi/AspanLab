  TrustControl — Установка на кассовый ПК
═══════════════════════════════════════════════

ПЕРВЫЙ РАЗ (делается один раз):

  Шаг 1.  Двойной клик на  1_SETUP.bat
          Подождите — установит Python и все программы автоматически.
          Не закрывайте окно до появления "Установка завершена!"

  Шаг 2.  Зайдите в личный кабинет:
          https://aspanlab-1.onrender.com
          → Раздел "Точки" → нажмите кнопку ⧉ рядом с API-ключом
            (ключ скопируется автоматически)

  Шаг 3.  Двойной клик на  2_CONFIG.bat
          Откроется Блокнот. Найдите строку:
            API_KEY=ВСТАВЬ_СЮДА_API_КЛЮЧ
          Выделите текст  ВСТАВЬ_СЮДА_API_КЛЮЧ  и удалите его.
          Правая кнопка мыши → "Вставить" — вставится ключ.
          Меню "Файл" → "Сохранить" → закрыть Блокнот.

  Шаг 4.  Двойной клик на  3_RUN.bat
          Мониторинг запустится. Не закрывайте чёрное окно.
          Если окно случайно закрылось — просто запустите 3_RUN.bat снова.

─────────────────────────────────────────────
АВТОЗАПУСК — чтобы мониторинг запускался сам при включении ПК:

  → Двойной клик на  АВТОЗАПУСК.bat  (делается один раз)

─────────────────────────────────────────────
КАЖДЫЙ ДЕНЬ:

  → Просто убедитесь что чёрное окно "TrustControl активен" открыто.
    Если нет — двойной клик на 3_RUN.bat

─────────────────────────────────────────────
ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

  1. Проверьте интернет (открывается ли браузер?)
  2. Убедитесь что микрофон подключён к компьютеру
  3. Запустите 1_SETUP.bat снова — он проверит все компоненты

ЛИЧНЫЙ КАБИНЕТ:
  → https://aspanlab-1.onrender.com

  TrustControl — Установка на кассовый ПК
═══════════════════════════════════════════════

ПЕРВЫЙ РАЗ (делается один раз):
  1. Скопируй всю папку на рабочий стол
  2. Двойной клик на  1_SETUP.bat  — установит Python и зависимости
  3. Двойной клик на  2_CONFIG.bat — откроется Блокнот
     → Найди строку API_KEY=
     → Удали ВСТАВЬ_СЮДА_API_КЛЮЧ
     → Вставь ключ из личного кабинета (Точки → скопировать API ключ)
     → Файл → Сохранить → Закрыть
  4. Двойной клик на  3_RUN.bat — запускает мониторинг

АВТОЗАПУСК (чтобы запускался сам при включении ПК):
  → Двойной клик на  АВТОЗАПУСК.bat

КАЖДЫЙ ДЕНЬ:
  → Просто двойной клик на  3_RUN.bat

# Образцы записей для STT-сводки

Положи сюда **2-5 коротких .wav** (свои тестовые записи кассы, не реальных клиентов —
ради приватности). По ним гоняется сводка:

- **локально:** `python scripts/hybrid_eval.py tests/samples`
- **в CI:** Actions → «STT eval (сводка по записям)» → Run workflow

Поддерживаются: `.wav .mp3 .m4a .ogg .webm`. Файлы маленькие — без них CI просто
напишет «нет аудио» и не упадёт.

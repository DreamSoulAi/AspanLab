# TrustControl — Маркетинг: парсер лидов + WhatsApp рассылка

Два скрипта чтобы за вечер собрать тёплых лидов в Алматы и начать диалоги.

## 1. Установка (один раз)

```bash
cd marketing
pip install -r requirements-marketing.txt
```
Для WhatsApp нужен установленный **Google Chrome** (chromedriver selenium ставит сам).

## 2. Собрать лиды из 2ГИС

1. Получи **бесплатный** ключ Catalog API на https://dev.2gis.ru
2. Задай ключ и запусти:

```bash
export DGIS_API_KEY=ваш_ключ        # Windows: set DGIS_API_KEY=ваш_ключ
python parse_2gis.py                 # кофейни, аптеки, фастфуд по Алматы
```

Результат → `leads.csv` (название, телефон, адрес, рейтинг, ссылка 2ГИС).
Отсортировано по рейтингу — сначала живые платёжеспособные точки.

Опции:
```bash
python parse_2gis.py --categories кофейня фастфуд --pages 30
python parse_2gis.py --city Астана
```

## 3. Разослать в WhatsApp

```bash
python whatsapp_outreach.py --dry-run    # посмотреть кому и что уйдёт
python whatsapp_outreach.py              # боевой режим (откроется Chrome + QR)
```

Первый запуск: откроется WhatsApp Web → отсканируй QR с телефона
(Настройки → Связанные устройства). Сессия сохранится, второй раз QR не нужен.

## ⚠️ Защита от бана — ОБЯЗАТЕЛЬНО прочитай

WhatsApp банит за массовый холодный спам. Встроенные защиты:
- не больше **35 сообщений за запуск** (`--limit`)
- случайные паузы **45-100 сек** между сообщениями
- **3 разных шаблона** сообщения (рандом) — не один и тот же текст
- **без ссылок** в первом касании (главный триггер спам-фильтра)
- лог `sent.csv` — повторно одному номеру не пишем

Твои действия чтобы номер выжил:
- бери **ОТДЕЛЬНЫЙ** номер, не личный
- первые дни — по 20-30 сообщений, потом наращивай
- **отвечай на ответы лично и быстро** — это главный сигнал «не спам»
- цель первого сообщения — не продать, а выйти на ЛПР и договориться о показе

# TrustControl — Roadmap к production-ready продукту

## Текущий статус: MVP ✅

Что готово и работает:
- Полная структура проекта (31 файл)
- Все API endpoints с авторизацией
- База данных со всеми моделями
- Анализ речи + детектор мошенничества
- Telegram уведомления
- Веб-дашборд
- Security: rate limiting, валидация, CORS
- Docker-compose для деплоя

---

## Слой 1 — БЭКЕНД

### Готово ✅
- FastAPI + SQLAlchemy async
- JWT авторизация с rate limiting
- 5 моделей БД
- Анализ фраз для 8 типов бизнеса
- Детектор мошенничества
- Whisper транскрипция

### Нужно доделать 🔧
- [ ] **Alembic миграции** — без них нельзя обновлять схему БД в продакшне
- [ ] **Логирование** — structlog или loguru, запись всех ошибок
- [ ] **Кеширование** — Redis для статистики дашборда (сейчас каждый раз из БД)
- [ ] **Фоновые задачи** — Celery или ARQ для итогов смен и ретраев
- [ ] **Пагинация** — курсорная пагинация для больших списков отчётов
- [ ] **Webhook Kaspi** — автоподтверждение оплаты
- [ ] **Email** — уведомления при мошенничестве через SMTP

### Файлы которые нужно создать
```
backend/
  core/
    logging.py        ← структурированные логи
    cache.py          ← Redis кеш
    exceptions.py     ← кастомные исключения
  tasks/
    shifts.py         ← фоновый итог смен в 14:00 и 22:00
    payments.py       ← проверка истёкших подписок
  database/
    migrations/       ← Alembic миграции
```

## Приоритеты для первого деплоя

Минимум что нужно перед первым клиентом:

1. `Dockerfile` + `nginx.conf` — деплой на VPS
2. Alembic миграции — безопасное обновление БД
3. `test_security.py` — убедиться что данные изолированы
4. Мобильная вёрстка дашборда — владельцы смотрят с телефона

Остальное можно добавлять итерационно после первых клиентов.

## Оценка трудозатрат

| Задача | Сложность | Время |
|--------|-----------|-------|
| Dockerfile + nginx | Низкая | 2-4 часа |
| Alembic миграции | Средняя | 4-8 часов |
| Базовые тесты | Средняя | 1-2 дня |
| Redis кеш | Средняя | 1 день |
| Мобильный дашборд | Средняя | 2-3 дня |
| CI/CD pipeline | Средняя | 1-2 дня |
| Полный фронтенд | Высокая | 1-2 недели |

**Итого до первого клиента:** ~1 неделя работы программиста
